    initialContent

    initialContent: unknown

    The initial document to load into the editor as a ProseMirror JSON object. Pass null or undefined to start with an empty document.

    Remarks

    Typed as unknown because the ProseMirror JSON schema is not exposed from Tiptap's public types and callers typically store it as opaque JSON.
