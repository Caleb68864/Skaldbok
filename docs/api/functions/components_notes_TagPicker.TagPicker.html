    • TagPicker(selected: TagPickerProps): Element

      Interactive tag selector used on note forms throughout the app.

      Parameters

      • selected: TagPickerProps

        Tags that are currently active (highlighted).

      Returns Element

      Remarks

      Renders a scrollable row of pill chips for all predefined tags plus any campaign-specific custom tags. Tapping a chip calls onToggle with the tag string; the caller owns the selection state.

      Below the chip row is a text input that lets users create new custom tags. Input is normalised to lowercase with spaces replaced by hyphens before being passed to onToggle (and optionally onCreateTag). If the typed value matches an existing predefined tag, the predefined chip is simply toggled instead of creating a new custom entry.

      Submission is possible via the + button or by pressing Enter while the input is focused.

      Example

      const [tags, setTags] = useState<string[]>([]);

      <TagPicker
        selected={tags}
        onToggle={tag =>
          setTags(prev =>
            prev.includes(tag) ? prev.filter(t => t !== tag) : [...prev, tag]
          )
        }
        customTags={campaign.customTags}
        onCreateTag={tag => addCustomTagToCampaign(tag)}
      />
