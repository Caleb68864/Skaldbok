      A sticky horizontal chip-row that lets the user pick which party member(s) an action applies to.

      When multiSelect is true (the default) a special "Party" chip appears (for groups of 2+) that selects or deselects all members at once. Duplicate member names are automatically disambiguated by appending a numeric rank suffix (e.g., "Raven (1)", "Raven (2)").
