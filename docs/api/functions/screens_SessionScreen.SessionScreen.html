    Function SessionScreen

    • SessionScreen(): Element

      The Session screen — the primary hub for managing the current play session.

      Returns Element

      The session management UI, or NoCampaignPrompt when no campaign has been selected.

      Remarks

      Displays the active campaign name and one of two views depending on whether a session is in progress:

      Active session view:

      • Session title and live elapsed-time counter (updated every 10 seconds).
      • End Session, Export Session (.md), and Export + Notes (.zip) actions.
      • SessionQuickActions panel for quick log entries.
      • Combat management: Start Combat / Resume Combat / CombatTimeline.

      No active session view:

      • Recap card showing the most recently ended session and its note count.
      • Start Session button.

      Shared elements:

      • Export All Notes (.zip) button.
      • Scrollable list of past (ended) sessions with per-session .md / .zip export buttons and a Resume button.
      • NotesGrid for all campaign notes.
      • EndSessionModal confirmation dialog.

      The component redirects to NoCampaignPrompt if no campaign is active.
