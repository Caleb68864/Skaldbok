## Intent

Provide a stateless, testable export layer that converts Dexie records to Obsidian-compatible markdown. No React dependency. Output quality is the killer-feature gate.

## Acceptance Criteria

| ID | Criterion |

### 1. Create `src/utils/export/generateFilename.ts`

```ts
export function generateFilename(note: { title: string; id: string }): string {
  const slug = note.title
    .toLowerCase()
    .replace(/[^a-z0-9\s-]/g, '')
    .trim()
    .replace(/\s+/g, '-');
  const idSuffix = note.id.slice(-6);
  return `${slug}-${idSuffix}.md`;
}
```

- Returns lowercase, hyphens, no special chars, with 6-char ID suffix
- If title is empty, fallback to `note-${idSuffix}.md`

### 2. Create `src/utils/export/resolveWikiLinks.ts`

```ts
// ProseMirrorNode type (minimal, no @tiptap import required)
type ProseMirrorNode = {
  type: string;
  text?: string;
  attrs?: Record<string, unknown>;
  content?: ProseMirrorNode[];
  marks?: Array<{ type: string; attrs?: Record<string, unknown> }>;
};

export function resolveWikiLinks(
  prosemirrorJson: unknown,
  allNotes: Array<{ id: string; title: string }>
): string { ... }
```

- Implement a minimal recursive serializer for node types the editor actually emits:
  - `doc` → serialize children
  - `paragraph` → serialize children + `\n\n`
  - `text` → return `node.text`
  - `mention` → find note in `allNotes` by `attrs.id`; if found return `[[Note Title]]`; if not found return `attrs.title` (plain text, no brackets)
  - `heading` → `#`/`##`/`###` prefix based on `attrs.level`
  - `bulletList` → serialize children as `- item\n`
  - `listItem` → serialize children
  - Unknown nodes → serialize content recursively (graceful fallback)
- If `prosemirrorJson` is null/undefined/not an object, return empty string

### 3. Create `src/utils/export/renderNote.ts`

```ts
export function renderNoteToMarkdown(
  note: Note,
  entityLinks: EntityLink[],
  allNotes: Array<{ id: string; title: string }>
): string { ... }
```

Steps:
1. Build YAML front matter object with fields: `title`, `type`, `id`, `campaignId`, `sessionId`, `createdAt`, `updatedAt`, `tags`
   - `tags` defaults to `[]` if not present on note
   - `sessionId` may be null — serialize as empty string or omit if null
2. Serialize front matter:
   ```
   ---
   title: "Note Title"
   type: generic
   id: abc123
   ...
   ---
   ```
3. Serialize body: call `resolveWikiLinks(note.body, allNotes)` to convert ProseMirror JSON → markdown
4. Return `frontMatter + '\n' + body`

Front matter YAML contract:
- String values containing `:` or `"` are quoted
- Multi-word values are quoted
- Arrays formatted as YAML lists

### 4. Create `src/utils/export/renderSession.ts`

```ts
export function renderSessionBundle(
  session: Session,
  linkedNotes: Note[],
  entityLinks: EntityLink[]
): Map<string, string> { ... }
```

Steps:
1. Build session index file:
   - Front matter: `title`, `id`, `campaignId`, `status`, `date`, `startedAt`, `endedAt`, `linkedNotes: [filename1, filename2, ...]`
   - Body: `## Notes\n` + list of note titles linking to their filenames
2. For each linked note, call `renderNoteToMarkdown(note, entityLinks, linkedNotes)`
3. Generate filenames with `generateFilename(note)` — deduplicate by appending `-2`, `-3` etc. if collision
4. Build Map: `{ [sessionFilename]: sessionMarkdown, [noteFilename]: noteMarkdown, ... }`
5. Return Map (no duplicate keys)

Session index filename: `generateFilename({ title: session.title, id: session.id })`

export function renderCampaignIndex(

### 6. Create `src/utils/export/bundleToZip.ts`

```ts
import JSZip from 'jszip';

export async function bundleToZip(files: Map<string, string>): Promise<Blob> {
  const zip = new JSZip();
  for (const [filename, content] of files) {
    zip.file(filename, content);
  }
  return await zip.generateAsync({ type: 'blob' });
}
```

- JSZip is pre-approved (no escalation needed)
- All files added at root level of zip

### 7. Create `src/utils/export/delivery.ts`

```ts
export async function shareFile(blob: Blob, filename: string): Promise<void> {
  if (navigator.canShare && navigator.canShare({ files: [new File([blob], filename)] })) {
    await navigator.share({ files: [new File([blob], filename)], title: filename });
  } else {
    downloadBlob(blob, filename);
  }
}

export async function copyToClipboard(markdown: string): Promise<void> {
  await navigator.clipboard.writeText(markdown);
}

export function downloadBlob(blob: Blob, filename: string): void {
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}
```

- `shareFile`: check `navigator.canShare` before calling `navigator.share` (AC-S5-08)
- `copyToClipboard`: uses `navigator.clipboard.writeText`
- `downloadBlob`: creates a temporary anchor element for file download
