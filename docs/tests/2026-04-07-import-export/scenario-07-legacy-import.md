# Scenario 07: Legacy .skaldmark.json Import

## Steps

1. Navigate to `/library`
2. Dismiss stale session warning if present
3. Click "Import Character"
4. Upload a sample file renamed to `.skaldmark.json` extension
5. Verify character imports successfully

## Expected

- Legacy format accepted
- Character imported with success toast

## Note

The character library file input accepts `.json` broadly, so legacy `.skaldmark.json` files will work. The bundle import on the Session screen explicitly accepts `.skaldmark.json` in its file picker.
