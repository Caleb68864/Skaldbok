# Scenario 02: Character Import via Library

## Steps

1. Navigate to `/library`
2. Dismiss stale session warning if present
3. Click "Import Character" button
4. Upload `sample-data/examples/viktor-vale.skaldbok.json`
5. Verify success toast appears
6. Verify character "Viktor Vale" appears in the library list

## Expected

- Character imported successfully
- Toast: "Character imported successfully"
- Character visible in library
